Point Wolfe River Watershed Delineation

Description:

This project performs a delineation of the Point Wolfe River watershed, located next to the Bay of Fundy in New Brunswick. Using LIDAR data provided by the New Brunswick GeoNB Data Catalogue and processed using the powerful LAStools.

The workflow for processing the Lidar data included; using LAStools to convert the data into smaller tiles, classifying point data as either ground or non-ground points, calculating the height of points above the ground, classifying calculated height values, and lastly generating both a digital terrain model (DTM) and digital surface model (DSM). Following the production and merger of the DTM tiles, the watershed delineation was performed using ArcGIS Hydrography Spatial Analyst tool set.

Project Requirements:
  • ArcGIS 10.7
  • LIDAR Data processing with Python, using;
    • GDAL
    • LAStools
  • New Brunswick GeoNB Data Catalogue Lidar Data
  • NRCan - 1945-2011 DEM data
  • CanVec Elevation Contour Line features
  • New Brunswick Hydrographic Network data
