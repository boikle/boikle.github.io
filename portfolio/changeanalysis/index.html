Temporal Change Analysis of a Volcano

Description:

The goal of this project was the detection of surface changes caused by the July 17th, 2001 eruption of Mt. Etna (located in Sicily, Italy). Landsat 7 imagery was selected before and after the eruption, with a temporal range of 16 days.

Using PCI Geomatica's EASI scripting language, the imagery was first prepared, and then temporal change analysis was performed using a variety of methods include Band Ratioing, NDVI difference analysis, SAVI difference analysis and Tasseled Cap Transformation.

Following the analysis, the results were examined in PCI Geomatica's Focus, then areas of volcanic change were identified, and an unsupervised classification was performed on those selected areas of change. The final result illustrated the locations of both volcanic activity, and areas changed by weather events or humans.

Project Requirements:
  • Landsat 7 Imagery (provided by http://glovis.usgs.gov/)
  • PCI Geomatica
  • Image analysis and processing using EASI scripts
  • Temporal change analysis using Band Ratioing, NDVI difference analysis, SAVI difference analysis, and Tasseled Cap Transformation, and a final classification of analysis results using an Unsupervised Classification
