Wildfire Analysis

Description:

This project performs an analysis on the 2017 Elephant Hill wildfire near Kamloops, British Columbia. Using only open source software to perform this analysis, Python scripts utilizing the GDAL and NumPy libraries calculated the normalized burn ratio (NBR) on Landsat 8 imagery, and QGIS was used to map the results.

The analysis began with NBR being calculated on imagery acquired the day prior to the wildfire starting, and also on imagery close to when the wildfire was contained. Following this analysis a difference between the two NBR results was then performed to determine the level of change to the landscape. Using QGIS, the analysis results were then classified based on burn severity using the classification provided here.

Project Requirements:
  • QGIS
  • Data Analysis with Python, using;
    • GDAL
    • NumPy
  • Landsat 8 Imagery
  • Normalized Burn Ratio
  • Change analysis
